Aspect Active Voice Passive Voice
Subject-Verb Order Subject does the action Subject receives the action
Verb Form Simple verb form "To be" + past participle
Focus On the doer of the action On the action or recipient
Sentence Structure Subject + Verb + Object Object + "To be" + Past Participle + (by Subject)
When to Use For clarity, directness, and emphasis on the doer When the action is more important than the doer, or when the doer is unknown

Rules for converting active voice to passive voice with respect to different tenses:

Tense Active Voice Rule Passive Voice Rule
Simple Present Subject + V1 is/am/are + V3
Simple Past Subject + V2 was/were + V3
Simple Future Subject + will + V1 will be + V3
Present Continuous Subject + is/am/are + V-ing is/am/are + being + V3
Past Continuous Subject + was/were + V-ing was/were + being + V3
Present Perfect Subject + has/have + V3 has/have + been + V3
Past Perfect Subject + had + V3 had + been + V3
Future Perfect Subject + will have + V3 will have been + V3
